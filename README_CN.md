
# music-tracker

Music 预售数据跟踪器

## 依赖

* 苹果操作系统
* [Textql][]
* Curl

# 使用

## `fetch`

获取每日竞投数据

```sh
# Fetch 11 Oct 2014 data
./fetch 11-10-2014
```

### `total`

统计

```sh
./total
```
