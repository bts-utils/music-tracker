
# music-tracker

BitShares Music Note pre-sale tracker.


## Dependences

* Mac OS X
* [Textql][]
* Curl


## Usage

### `fetch`

Fetch each day data


```sh
# Fetch 11 Oct 2014 data
./fetch 11-10-2014
```

### `total`

```sh
./total
```
